Getting started

1. Onboard your primary vendor distributions on the Suppliers management tab.

2. Register specific inventory line instances against those vendor IDs on the Products configuration catalog.

3. Process cryptographic ledger checkpoint transactions at transfer nodes through the Track Shipment logging gateway.

4. Consumers and validation endpoints can instantly confirm item provenance using the public verification interface tool on the Verify Product dashboard utility.
